## Visual Markup Chrome Extension

This extension allows you to:

* See your markup clearly

* Diagnose CSS issues

* Understand element positioning 

* View the names of classes and ids

Get the extension on the [Chrome Webstore](https://chrome.google.com/webstore/detail/visual-markup/haplgakkgklldgnckfdapejhipchiceo).

To use, simply click the Visual Markup icon. 

Click again to remove the visual markup.
